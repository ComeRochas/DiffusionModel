# Results Directory

This directory will contain generated images:
- `sample_unconditional.png`: Unconditional samples without guidance
- `sample_guided.png`: Class-conditional samples with classifier guidance
- `counterfactual_example.png`: Counterfactual pairs demonstrating causal intervention
- `all_digits_guided.png`: Grid of all 10 digit classes

Run `sample_counterfactuals.py` after training to generate these images.
